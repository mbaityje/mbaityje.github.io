# mbaityje.github.io
Personal website made by following http://jmcglone.com/guides/github-pages/

## Local Preview
Use Jekyll to preview the site locally:

```bash
/opt/homebrew/lib/ruby/gems/4.0.0/bin/jekyll serve --safe --host 127.0.0.1 --port 4002
```

Then open `http://127.0.0.1:4002` in your browser.
